# Battlefield 6 REDSEC map item locations

## View the map [HERE](https://d00msdaydevice.github.io/REDSEC/)

This map was made with gdal-bin version (3.6.2+dfsg-1+b2)
GDAL 3.6.2, released 2023/01/02

### Map Tile Generation & Tooling:
Tiles were generated using GDAL’s gdal2tiles.py script.

### Point of Interest (POI) System
POIs are rendered using MapLibre GL JS symbol layers and loaded from a GeoJSON file.

### Icon System:
Icons are loaded dynamically from the /icons directory

### Map Rendering Library:
MapLibre GL JS
Version: 3.6.0
Source: https://maplibre.org/
